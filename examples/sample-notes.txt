1. 文件整理工具的目標

這個作品希望把原本分散在 PDF、Word 或純文字檔案裡的內容，轉換成容易閱讀的 HTML 學習筆記。使用者不需要手動排版，只要上傳檔案，就可以取得章節、快速重點和可列印的頁面。

2. 技術方法

後端使用 Python 解析檔案並抽取文字。PDF 解析引用 pypdf，Word 解析引用 python-docx。前端使用原生 HTML、CSS 和 JavaScript 完成拖放上傳、結果預覽、HTML 原始碼顯示、下載和列印。

3. 可以展示的能力

這個專案可以展示我會整合開源套件、設計資料處理流程、建立 API，並把處理結果包成可閱讀的網站介面。教授可以直接看到輸入文件到輸出 HTML 的完整流程。
